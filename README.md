DataGenerator
==============

Takes as input
* number of rows
* number of columns
* types of columns
* type of data gen

Column Types
* ranges for random numbers
* ranges for dates
* strings

String subtypes
* first name
* last name
* state
* city

Type of data gen
* pick from a list at random
* pick next value from list
* repeating (string, date, or number)
* incrementing (date or number)
